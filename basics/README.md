# Python Basics

## Purpose
This folder contains beginner‑level Python practice files that cover the core fundamentals of the language. These scripts are part of my daily learning plan and help build a strong foundation for future projects and tools.

## Topics Covered
- Variables and data types  
- Strings and string methods  
- Integers, floats, and basic math  
- Booleans and comparisons  
- Input and output  
- If/else statements  
- For loops and while loops  
- Basic script structure and printing  

## How to Run Scripts
1. Open a terminal in this folder  
2. Run any script using Python:

python3 filename.py


(Replace `filename.py` with the actual file you want to run.)

If you're on Windows and `python3` doesn’t work, use:

python filename.py
